# Retail Business Analysis Project

## Overview
This project analyzes retail sales data to provide actionable business insights.  
It includes SQL Server and MongoDB connections, visualizations, and data analysis.

## Features
- Top Customer Analysis  
- Branch Revenue (Last Quarter)  
- Daily Sales Trends  
- Branches Ranked by Revenue  
- Ramadan vs Other Sales Comparison  
- Top Product Performance  
- Monthly Revenue Over Time
